# M-TOOLS: Multilingual Tool Calling Benchmark

Benchmarking Robust Multilingual Tool Calling Under Code-Switching and Parameter Localization.

## Repository Structure

- `data/`: Contains the dataset in various stages (raw, cleaned, validated, etc.)
- `tools/`: Tool schemas and mock execution environment
- `evaluation/`: Evaluation pipeline and metrics
- `generation/`: Scripts for dataset expansion and generation
- `scripts/`: General utility scripts
- `experiments/`: Experiment configurations and outputs
- `results/`: Final results and failure analysis
- `paper/`: Drafts and final EMNLP paper materials
